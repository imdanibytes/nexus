## Auto-Versioning

The release workflow now automatically syncs version fields from the git tag across all config files (`package.json`, `tauri.conf.json`, `Cargo.toml`, and the Plugin SDK). No more manual version bumps during releases.

Fixed the SDK publish step failing with 409 Conflict by auto-deriving the SDK version from the release tag.
